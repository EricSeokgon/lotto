# SPEC-LOTTO-080 인수 기준 (Acceptance Criteria)

## 손계산 검증 픽스처 (4개 회차)

| 회차 | 본번호 | 정렬 간격 | max_gap | 구간 |
|------|--------|-----------|---------|------|
| D1 | [1,2,3,4,5,6] | [1,1,1,1,1] | 1 | "1-5" |
| D2 | [1,10,20,30,40,44] | [9,10,10,10,4] | 10 | "6-10" |
| D3 | [1,2,3,40,41,42] | [1,1,37,1,1] | 37 | "31+" |
| D4 | [5,10,15,20,25,30] | [5,5,5,5,5] | 5 | "1-5" |

- max_gap_distribution: "1-5"=2(50.0%), "6-10"=1(25.0%), "11-15"=0, "16-20"=0, "21-30"=0, "31+"=1(25.0%)
- avg_max_gap = (1+10+37+5)/4 = 13.25
- most_common_range = "1-5" (2회 최빈)
- high_gap_pct = max_gap>=21 인 D3 1건/4건 = 25.0

## 경계 / 빈 입력

- **AC-01**: 빈 draws → total_draws=0.
- **AC-02**: 빈 draws → avg_max_gap=0.0.
- **AC-03**: 빈 draws → high_gap_pct=0.0.
- **AC-04**: None 입력도 예외 없이 빈 구조 반환.
- **AC-05**: 빈 draws → most_common_range="1-5".

## 버킷 헬퍼 (_max_gap_bucket)

- **AC-06**: g=1, g=5 → "1-5".
- **AC-07**: g=6, g=10 → "6-10".
- **AC-08**: g=11, g=15 → "11-15".
- **AC-09**: g=16, g=20 → "16-20".
- **AC-10**: g=21, g=30 → "21-30".
- **AC-11**: g=31, g=44 → "31+".

## max_gap 계산 (단일 회차)

- **AC-12**: [1,2,3,4,5,6] → max_gap=1 → "1-5" count=1.
- **AC-13**: [1,2,3,40,41,42] → max_gap=37 → "31+" count=1.
- **AC-14**: [5,10,15,20,25,30] → max_gap=5 → "1-5" count=1.
- **AC-15**: [1,10,20,30,40,44] → max_gap=10 → "6-10" count=1.
- **AC-16**: 보너스 번호는 계산에 포함되지 않는다.

## 응답 구조

- **AC-17**: 반환 dict는 5개 최상위 키를 모두 포함한다.
- **AC-18**: max_gap_distribution은 항상 6개 고정 키만 포함한다.
- **AC-19**: 각 분포 항목은 count·pct 키를 가진다.
- **AC-20**: 모든 버킷 count 합은 total_draws와 같다.
- **AC-21**: pct는 소수 2자리로 반올림된다.

## 파생 지표 (픽스처)

- **AC-22**: avg_max_gap = 13.25.
- **AC-23**: most_common_range = "1-5".
- **AC-24**: high_gap_pct = 25.0.
- **AC-25**: most_common_range 동률 시 더 작은(앞선) 구간이 선택된다.

## 캐시

- **AC-26**: 동일 입력 재호출 시 캐시된 동일 객체 반환.
- **AC-27**: invalidate_cache 후 새 결과 객체 생성(값은 동일).
- **AC-28**: invalidate_cache가 _max_gap_dist_cache를 비운다.

## 라우트

- **AC-29**: GET /api/stats/max_gap_dist → 200 + 키 구조.
- **AC-30**: GET /api/stats/max_gap_dist 은 데이터 없어도 200.
- **AC-31**: GET /stats/max-gap-dist → 200(HTML).
- **AC-32**: GET /stats/max-gap-dist 은 데이터 없어도 200.
